## Developer instructions

### Build OfflineApp

To develop offlineApp (Vite + Vue app + PWA), you can use within OfflineApp/src
```
vite
```

To build offlineApp, you need to cd inside OfflineApp/src :
```
npm run build
npm run copy
```
